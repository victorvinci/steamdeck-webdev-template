# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)** as its code of conduct. The canonical text lives at the link above and governs all project spaces — issues, pull requests, discussions, and any other channel tied to this repository.

## Scope

The Covenant applies to anyone participating in this project: maintainers, contributors, reviewers, and one-off commenters. It also applies when someone represents the project in public — e.g. a talk, a social-media post tagging the repo, or a comment on a downstream fork that links back here.

## Reporting

Report conduct concerns privately to the maintainer at **victorvinci@protonmail.com**. Reports are handled confidentially; the maintainer will acknowledge within 72 hours.

If the concern is a security vulnerability rather than a conduct issue, use the flow in [`SECURITY.md`](./SECURITY.md) instead.

## Enforcement

The maintainer follows the **Enforcement Guidelines** section of the Covenant (also at the link above) when deciding on a response. Possible outcomes range from a private correction to a permanent ban from project spaces, scaled to the severity and pattern of the conduct.

## Downstream forks

This is a fork-first template. Downstream forks are encouraged to keep a code of conduct in place, but own their own enforcement. Nothing in this file binds a fork's maintainers.
